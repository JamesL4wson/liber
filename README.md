# liber

An R^n to R^n graphing calculator written in C++. It aims to be able to generate any mathematical visualalization where data is presented as locations within a space. 

The calculator can handle, explicit and implicit functions, parametric functions, ODEs, PDEs and vector valued functions. Furthermore, liber can parse many 3D file formats and handle data from a number of spreadsheet formats. 

The input and output dimensions of any plot can be any size and even high-dimensional plots can be generated as 3D cross-sections or projections. When plotting cross-sections, liber's speed is independant of the number of dimensions, allowing it to maintain rendering speeds for even high-dimensional spaces.
